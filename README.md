# Listar-Eventos-e-Baixar-Todos-eTCESP
Extensão do TamperMonkey com as seguintes funcionalidades: 1) Listar o número dos eventos ao lado do nome do arquivo, evitando ter que clicar em "Navegar Pelo Processo". 2) Cria um botão que permite baixar automaticamente todos os arquivos do TC; os arquivos são baixados no formato [Número EV + Nome do arquivo]
